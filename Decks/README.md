# Back 4 Blood - Deck Builds

This folder contains optimized deck builds for Back 4 Blood.

## Available Decks

- **MELEE_NIGHTMARE_DECK.txt** - Optimized melee deck for Nightmare difficulty multiplayer
  - 15-card deck focused on high damage, trauma resistance, sustain, and team support
  - Includes full card breakdown, buffs summary, and playstyle guide

## Deck Format

Each deck file includes:
- Complete card list with explanations
- Total buffs summary
- Playstyle guide
- Strategy tips
- Alternative/substitution options

## Source

All decks are built using cards from the [Card References](../Card%20References/) folder, which contains all 191 Active Cards from Back 4 Blood.
