# Back 4 Blood Active Cards Reference

This folder contains all 191 Active Cards from Back 4 Blood, organized by category and expansion.

## Structure

- **Reflex/** - 58 Reflex (Blue) cards - Focus on agility and speed enhancements
- **Discipline/** - 54 Discipline (Red) cards - Focus on accuracy and stamina improvements  
- **Brawn/** - 48 Brawn (Green) cards - Focus on health and resilience boosts
- **Fortune/** - 31 Fortune (Yellow) cards - Focus on resource management and luck-based benefits
- **Expansions/** - Cards from expansion packs:
  - Tunnels of Terror
  - Children of the Worm
  - River of Blood (8 cards)

## Total Cards: 191

- 58 Reflex cards
- 54 Discipline cards
- 48 Brawn cards
- 31 Fortune cards

## Card File Format

Each card is stored as a `.txt` file with the following format:

```
Card Name: [Name]
Category: [Category]
Type: [Type]
Affinity: [Affinity]

Description:
[Full card description including Campaign, Swarm, and other mode-specific effects]
```
